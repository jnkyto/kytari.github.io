EpstoriClient

EpstoriClient is a utility mod for Minecraft 1.16.5.
It features client-side commands for changing
brightness and fov beyond the limits of game settings.
Discord Rich Presence is also built-in,
which can be toggled on or off.

At this time the mod does not provide
any "illegal" in-game advantages. However,
because this mod is aimed at anarchy servers,
those kind of features will probably be implemented.

EpstoriClient is open source, currently licenced under CC0-1.0.
More info on GitHub.com
